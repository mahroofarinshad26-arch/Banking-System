Banking System Project

This is a simple Banking System program created using Python.

Description

This program allows users to perform basic banking operations like login, deposit, and withdrawal.

Features

- Login using username and password
- Deposit money
- Withdraw money
- Check account balance

How to Run

- Open the Python file
- Run the program
- Enter login details
- Choose an option from the menu
